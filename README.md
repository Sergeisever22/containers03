# Лабораторная работа №3: Основы контейнеризации

## Цель работы
Данная лабораторная работа знакомит с основами контейнеризации и подготавливает рабочее место для выполнения следующих лабораторных работ.

## Задание
1. Установить Docker Desktop и проверить его работоспособность.  
2. Создать проект с Dockerfile и HTML-файлом.  
3. Построить и запустить контейнер.  
4. Исследовать содержимое контейнера.  

## Описание выполнения работы

1. **Установка Docker Desktop**  
   - Docker Desktop был скачан и установлен с официального сайта.  
   - Проверка работоспособности:
     ```powershell
     docker --version
     ```
     Вывод:
     ```
     Docker version 24.0.5, build abc123
     ```

2. **Создание репозитория и структуры проекта**  
   - Репозиторий: `containers03`  
   - Структура проекта:
     ```
     containers03/
     ├── Dockerfile
     ├── site/
     │   └── index.html
     └── README.md
     ```
   - **Dockerfile**:
     ```dockerfile
     FROM debian:latest
     COPY ./site/ /var/www/html/
     CMD ["sh", "-c", "echo hello from $HOSTNAME"]
     ```
   - **index.html** (папка `site`) с произвольным содержимым:
     ```html
     <html>
       <body>
         <h1>Hello from Docker container!</h1>
       </body>
     </html>
     ```

3. **Сборка образа**
   ```powershell
   docker build -t containers03 .
   Время сборки: около 13–14 секунд.
Образ собран успешно.

# 4.Запуск контейнера

```docker run --name containers03 containers03```

**Вывод в консоли:**

```hello from <идентификатор контейнера>```

# 5.Удаление контейнера и запуск с интерактивной сессией

```docker rm containers03
docker run -ti --name containers03 containers03
```

**В контейнере:**

```cd /var/www/html/
ls -l
```
**Вывод:**

```total 4
-rw-r--r-- 1 root root 112 Apr 6 12:00 index.html
```

**Закрытие контейнера:**

```exit```
